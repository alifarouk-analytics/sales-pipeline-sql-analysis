# Sales Pipeline & Revenue Analysis

## Overview
This project analyzes sales opportunity data to evaluate revenue performance, pipeline efficiency, and conversion rates.

## Dataset
CRM sales opportunities dataset including deal stages, sales agents, and revenue.

## Key Insights
- Calculated total revenue and deal volume
- Analyzed win rate and average deal size
- Identified top-performing sales agents
- Evaluated pipeline distribution across deal stages

## Tools Used
- SQL

## Business Impact
These insights can help optimize sales strategy, improve conversion rates, and forecast revenue more effectively.
